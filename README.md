energozvit
==========

Energozvit - це програма для контролю спожитої електричної енергії та 
автоматичного створення і друку звітів. Робота з програмою полягає в веденні
щомісячних показників лічильників та друку відповідних звітів.
Дані зберігаються в базі даних sqlite. Створення pdf виконується з допомогою LaTeX.
